Main job system macros encapsulating interactions for multithreading.

◆ JADD [1/2]

#define JADD (   f)    vgjs::JobSystem::getInstance()->addJob( [=](){ f; } )

Add a function as a job to the jobsystem.

In multithreaded operations, this macro adds a function as job to the job system. In singlethreaded mode, this macro simply calls the function.

Parameters
[in]fThe function to be added or called.
