Test whether the projection of a point to a quad plane intersects with the quad.

The point does not have to be on the quad plane. Instead it is projected onto the quadplane first. Then it is tested whether the projection intersects with the quad.

Parameters
[in]pPoint in 3D space
[in]qQuad consisting of 4 vertex points
Returns
whether the point projection to the quad plane intersects with the quad
