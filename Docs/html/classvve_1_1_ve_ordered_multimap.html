Detailed Description

template<typename K, typename I>
class vve::VeOrderedMultimap< K, I >

A multi map storing key-value pairs, ordered with an AVL tree.

An AVL tree is a binary search tree. It starts with a root, and every node can have a left and a right successor (or both or none). AVL tress are always perfectly balanced. When ever e new node is inserted or deleted, it checks whether the balance has been compromised, and then rebalances by one or several subtree rotations. See https://www.geeksforgeeks.org/avl-tree-set-1-insertion/
