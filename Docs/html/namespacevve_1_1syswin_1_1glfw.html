void vve::syswin::glfw::key_callbackGLFW ( GLFWwindow *  window,
int  key,
int  scancode,
int  action,
int  mods

Callback to receive GLFW events from the keyboard.

This callback is called whenever the user does something the keyboard. Pressing a key will actually cause two events. One indicating that the key has been pressed, and another one indicating that the key is still pressed. The latter one is continuous until the key is released again. Also it will not be processed immediately but only in the following render loop, hence using the loop counter to schedule it for the future.

Parameters
[in]windowPointer to the GLFW window that caused this event
[in]keyKey ID that caused this event
[in]scancodeNot used
[in]actionID of what happened (press, release, continue)
[in]modsNot used

Callback to receive GLFW events from mouse buttons.

This callback is called whenever the user does something the mouse buttons. Pressing a mouse button will actually cause two events. One indicating that the button has been pressed, and another one indicating that the button is still pressed. The latter one is continuous until the button is released again. Also it will not be processed immediately but only in the following render loop, hence using the loop counter to schedule it for the future.

Parameters
[in]windowPointer to the GLFW window that caused this event
[in]buttonID of the mouse button that was interacted with
[in]actionID of the action (press, release, repeat)
[in]modsNot used
